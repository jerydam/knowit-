## ğŸš€ Getting Started

### Prerequisites

  * Node.js 18+ installed.
  * An [OpenAI API Key](https://platform.openai.com/).
  * A [Supabase](https://supabase.com/) project.
  * A [WalletConnect Cloud](https://cloud.walletconnect.com/) Project ID.

### 1\. Clone the Repository

```bash
git clone https://github.com/jerydam/knowit-.git
cd knowit-
```

### 2\. Install Dependencies

```bash
npm install
# or
yarn install
```

### 3\. Configure Environment Variables

Create a `.env.local` file in the root directory and add the following:

```env
# Blockchain
NEXT_PUBLIC_WALLET_CONNECT_PROJECT_ID=your_walletconnect_id
NEXT_PUBLIC_QUIZ_CONTRACT_ADDRESS=0xYourDeployedContractAddress

# Database (Supabase)
NEXT_PUBLIC_SUPABASE_URL=your_supabase_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_supabase_key
SUPABASE_SERVICE_ROLE_KEY=your_service_role_key

# AI & IPFS
OPENAI_API_KEY=sk-your_openai_key
PINATA_JWT=your_pinata_jwt
```

### 4\. Run Locally

```bash
npm run dev
```

Open `http://localhost:3000` in your browser.

## ğŸ“± Testing with MiniPay (Android)

Since MiniPay runs inside the Opera Mini browser on Android, you cannot test it strictly on localhost.

1.  **Start your local server:** `npm run dev`
2.  **Expose localhost to the web:** Use Ngrok.
    ```bash
    npx ngrok http 3000
    ```
3.  **Copy the HTTPS URL** generated by Ngrok (e.g., `https://xyz-123.ngrok-free.app`).
4.  **Open Opera Mini** on your Android device.
5.  **Navigate to the URL** inside the MiniPay wallet browser.
6.  **Verify:** The app should automatically detect MiniPay and show "ğŸ“± MiniPay Connected".
